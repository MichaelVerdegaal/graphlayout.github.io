Course overview

Graph layout algorithms, from theory to drawing

Start with the graph theory you actually need, then learn the five layout algorithms that get used in real visualizations. Each lesson has a live canvas demo, Python reference code, flashcards, and a knowledge check.

This course is for engineers who want to understand layouts well enough to implement them, debug them, or pick the right one for a given graph. Maths shows up where it earns its place and not before. Each lesson takes roughly 10 to 15 minutes.
